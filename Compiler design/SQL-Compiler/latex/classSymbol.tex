\doxysubsection{Detailed Description}
Base class representing a symbol in the grammar (terminal or non-\/terminal). 

The \doxylink{classSymbol}{Symbol} class encapsulates a string representation of a grammar symbol and a flag indicating whether it is a terminal or non-\/terminal symbol.
