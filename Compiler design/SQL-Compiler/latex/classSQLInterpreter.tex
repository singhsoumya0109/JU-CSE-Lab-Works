\doxysubsection{Detailed Description}
A class that interprets SQL queries using a lexer and parser. 

The \doxylink{classSQLInterpreter}{SQLInterpreter} class orchestrates the tokenization and parsing of SQL queries. It uses a \doxylink{classLexer}{Lexer} to break down input strings into tokens and a \doxylink{classParser}{Parser} to analyze the token sequence, with output controlled by a verbosity level.

Initializes and runs the SQL interpreter. 

Starts the interpretation process, prompting for input, tokenizing it, and parsing it according to the configured verbosity level.

Runs an interactive loop that\+:
\begin{DoxyItemize}
\item Prints parser details (grammar, first/follow sets, LL1 table) if verbosity is 2.
\item Prompts for SQL input, tokenizes it, and parses it until "{}exit"{} is entered.
\item Displays tokens if verbosity is 2.
