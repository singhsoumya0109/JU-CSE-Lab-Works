\doxysubsection{Detailed Description}
Implementation of the \doxylink{classLexer}{Lexer} class for tokenizing SQL query strings. 

This file contains the definitions of the \doxylink{classLexer}{Lexer} class methods, which process input strings and convert them into a sequence of \doxylink{classToken}{Token} objects for use in an LL1 parser for SQL queries.
