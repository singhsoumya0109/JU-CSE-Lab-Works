\doxysubsection{Detailed Description}
A class that tokenizes SQL query strings into a vector of \doxylink{classToken}{Token} objects. 

The \doxylink{classLexer}{Lexer} class takes an input string (e.\+g., an SQL query) and breaks it down into individual tokens, such as keywords, identifiers, operators, and literals. It uses helper methods to classify each token based on predefined rules and stores the results in a vector.
