\doxysubsection{Detailed Description}
A class implementing an LL1 parser for SQL queries. 

The \doxylink{classParser}{Parser} class loads a grammar from a file, computes the FIRST and FOLLOW sets, constructs an LL1 parsing table, and uses it to parse a sequence of tokens generated from an SQL query.
