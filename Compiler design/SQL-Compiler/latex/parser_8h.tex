\doxysubsection{Detailed Description}
Header file for the \doxylink{classParser}{Parser} class and related symbol classes for LL1 parsing of SQL queries. 

This file defines the \doxylink{classSymbol}{Symbol}, \doxylink{classNonTerminal}{Non\+Terminal}, \doxylink{classTerminal}{Terminal}, and \doxylink{classParser}{Parser} classes, which together implement an LL1 parser for SQL queries using a grammar loaded from a file.
