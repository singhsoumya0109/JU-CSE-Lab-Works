Detailed Description

A class representing a single token in an SQL query.

The Token class stores the type (e.g., "KEYWORD", "IDENTIFIER") and lexeme (the actual string) of a token, providing accessors to retrieve these values. It is used by the Lexer and Parser in an LL1 SQL interpreter.
