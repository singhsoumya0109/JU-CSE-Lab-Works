Detailed Description

A class that interprets SQL queries using a lexer and parser.

The SQLInterpreter class orchestrates the tokenization and parsing of SQL queries. It uses a Lexer to break down input strings into tokens and a Parser to analyze the token sequence, with output controlled by a verbosity level.

◆ init_interpreter()

void SQLInterpreter::init_interpreter ( )

Initializes and runs the SQL interpreter.

Starts the interpretation process, prompting for input, tokenizing it, and parsing it according to the configured verbosity level.

Runs an interactive loop that:

  • Prints parser details (grammar, first/follow sets, LL1 table) if verbosity is 2.
  • Prompts for SQL input, tokenizes it, and parses it until "exit" is entered.
  • Displays tokens if verbosity is 2.
